## About

> Tool to help format AO3 downloads for print

This Chrome extension (other formats coming soon) allows you to format the HTML downloads from Archive of Our Own for neater printing.

I suppose it should actually be called HTML of Our Own...?

### Current features:

- Page break after the tags, summary, and each chapter
- Adjusted font and colours to resemble print books
- Footer with page numbers

### Planned features:

- Customize the fonts and colours
- Better formatting of title and tags
- Add a cover image?

## Installation

Not available on Webstore yet. Please download the folder `eo3-extension`, then turn on Developer Mode in Chrome Extensions and load it as an unpacked extension.

## Usage

1. Navigate to an AO3 work.
2. Click the extension icon, then click the button in the popup.
3. Download should begin shortly.
